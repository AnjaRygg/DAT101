    4. [Encapsulation is a key concept in object-oriented programming (OOP) where we group data and methods into a class and limit direct access to some parts. This protects an object’s internal state by blocking unauthorized or accidental changes. By hiding certain details, we ensure that data can only be accessed or changed in a controlled way.]

    5. [When we use the new keyword in JavaScript, it sets up a new, empty object in memory that isn’t connected to anything else yet.]
